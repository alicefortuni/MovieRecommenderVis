Welcome to MovieRecommenderVis

The primary goal of this project is to deliver a simple and intuitive graphical interface that enables users to interact with data on users, movies, and recommendations derived from the renowned MovieLens dataset. This interface offers predefined database query functionalities, eliminating the need to manually write Cypher queries.
